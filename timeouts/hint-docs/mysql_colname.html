Funcoes de Banco de Dados (MySQL)
mysql_colname( object result, int colnum )
Retorna o nome da coluna no resultado da consulta.
Argumentos/Retorno:
  • result - Resultado de consulta no banco de dados
  • colnum - Número da coluna cujo nome será obtido
  • return - Nome da coluna no resultado da consulta
Exemplo:
object result = ...
string coluna = mysql_colname( result, 2 )
