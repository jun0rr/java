Funcoes de Ambiente Gráfico
uigettext( object comp )
Retorna o texto exibido por um elemento gráfico de texto.
Argumentos/Retorno:
  • comp - Elemento gráfico de texto
Exemplo:
object label = uilabel( "Texto do label" )
string texto = uigettext( label )
print( texto )
# Imprime no console
> Texto do label
