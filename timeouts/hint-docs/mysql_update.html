Funcoes de Banco de Dados (MySQL)
mysql_update( object conn, string sql )
Executa um comando SQL de atualização no banco de dados, retornando o número de linhas afetadas na atualização.
Argumentos/Retorno:
  • conn - Objeto de conexão com o banco de dados
  • sql - Comando SQL de atualização
  • conn - Número de linhas afetadas no comando de atualização
Exemplo:
object conn = ...
string sql = "UPDATE tabela SET campo = valor WHERE condicoes"
int linhas = mysql_update( conn, sql )
