Funcoes de Banco de Dados (MySQL)
mysql_connect_url( string url, string user, string pwd )
Inicia uma conexão com servidor MySQL utilizando a URL completa informada, autenticando com o usuário e senha fornecidos.
Retorna o objeto Connection que representa a conexão estabelecida.
Argumentos/Retorno:
  • url - Endereço URL compelto do servidor
  • user - Usuário do banco de dados
  • pwd - Senha de autenticação
  • return - Objecto Connection que representa a conexão
Exemplo:
# Inicia uma conexão na máquina local
string url = "jdbc:mysql://localhost:3306/schema"
object conn = mysql_connect_url( url, "usuario", "senha" )
