Funcoes de Texto
strsize( string arg )
Retona o tamanho do texto (string) informado.
Argumentos/Retorno:
  • arg - Texto
  • return - Numero de caracteres no texto
Exemplo:
string txt = "um23456789"
int tam = strsize( txt )
print( "TXT possui " + tam + " caracteres" )
# Imprime no console
> TXT possui 10 caracteres
