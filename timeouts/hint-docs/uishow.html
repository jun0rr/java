Funcoes de Ambiente Gráfico
uishow( object frame, int x, int y )
Exibe na tela uma janela previamente criada.
Argumentos/Retorno:
  • frame - Janela a ser exibida
  • x - Ponto horizontal na tela onde a janela será exibida (opcional)
  • y - Ponto vertical na tela onde a janela será exibida (opcional)
Exemplo:
object janela = uiframe( "Titulo da Janela" )
uisetsize( janela, 200, 100 )
uishow( janela )
