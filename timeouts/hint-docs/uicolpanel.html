Funcoes de Ambiente Gráfico
uicolpanel( int hgap, string align )
Cria um painel para agregação e organização de outros elementos gráficos. Os elementos adicionados são organizados em colunas dispostas de forma horizontal, um após o outro de acordo com a ordem que forem adicionados.
Argumentos/Retorno:
  • hgap - Distância horizontal entre os elementos, em pixels
  • align - Forma de alinhamento dos elementos dentro do painel (LEFT/CENTER/RIGHT)
Exemplo:
object label = uilabel( "Nome:" )
object text = uitext( "", 150 )
object cpanel = uicolpanel( 5, UILEFT )
uiadd( cpanel, label )
uiadd( cpanel, text )
