Funcoes SISBB
gettext( int lin, int col, int comp )
Captura no terminal e retorna o texto posicionado na linha e coluna especificados, com o comprimento comp.
Argumentos/Retorno:
  • lin - Linha da posicao do terminal
  • col - Coluna da posicao do terminal
  • comp - Numero de caracteres a serem retornados
  • return - O texto capturado do terminal
Exemplo:
string txt = gettext( 1, 3, 10 )
print( txt )
