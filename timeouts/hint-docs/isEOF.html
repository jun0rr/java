Funcoes de Leitura/Escrita (I/O)
isEOF( object bytes )
Verifica se a coleção de bytes informada representa o fim de arquivo.
Argumentos:
  • bytes - Coleção de bytes previamente lida de um arquivo.
Exemplo:
string arquivo = "c:/dados.zip"
string copia = "c:/dados_backup.zip"

object bytes = frbytes( arquivo )
while ! isEOF( bytes )
   fwbytes( copia, bytes )
   bytes = frbytes( arquivo )
endwhile

frclose( arquivo )
fwclose( copia )
