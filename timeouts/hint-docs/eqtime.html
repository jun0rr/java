Funcoes de Data
eqtime( object date1, object date2 )
Verifica se duas datas sao iguais, inclusive horas, minutos e segundos, retornando verdadeiro ou falso.
Argumentos/Retorno:
  • date1 - Primeira data
  • date2 - Segunda data
  • return - Verdadeiro/false
Exemplo:
object agora = date( "01/04/2014 12:00:00" )
object hoje = date( "01/04/2014 12:00:00" )
if eqtime( agora, hoje )
    print( agora + " = " + hoje )
endif
# Imprime no console
> 01/04/2014 12:00:00 = 01/04/2014 12:00:00
