Funcoes de Ambiente Gráfico
uiframe( string title, string onclose )
Cria uma janela gráfica vazia, onde podem ser adicionados elementos para interação com o usuário.
Argumentos/Retorno:
  • title - Título da janela
  • onclose - Função a ser chamada no fechamento da janela
Exemplo:
object frame = uiframe( "My Frame" )
