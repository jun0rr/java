Funcoes de Data
getday( object arg )
Retorna o dia do mes da data informada, ou a quantidade de dias da diferenca de tempo.
Argumentos/Retorno:
  • arg - Data ou diferenca de tempo
  • return - Dia do mes ou quantidade de dias
Exemplo:
object data = date( "15/04/2014" )
object hoje = date( "01/04/2014" )
object tempo = diff( hoje, data )
print( getday( data ) )
print( getday( tempo ) )
