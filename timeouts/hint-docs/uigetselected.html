Funcoes de Ambiente Gráfico
uigetselected( object comp )
Retorna o elemento selecionado em um elemento de seleção.
Argumentos/Retorno:
  • comp - Elemento gráfico de seleção
Exemplo:
object combo = uicombo( "", "one", "two", "ten" )
uisetselected( combo, "two" )
object select = uigetselected( combo )
print( select )
# Imprime no console
> two
