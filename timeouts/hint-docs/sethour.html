Funcoes de Data
sethour( object date, int hour )
Define a hora da data informada, retornando a mesma.
Argumentos/Retorno:
  • data - Data
  • hour - Hora do dia a ser definida
  • return - Data com a hora definida
Exemplo:
object data = date( "15/05/2014" )
data = sethour( data, 8 )
print( data )
# Imprime no console
> 15/05/2014 08:00:00
