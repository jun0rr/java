Funcoes de Data
getyear( object arg )
Retorna o ano da data informada, ou a quantidade de anos da diferenca de tempo.
Argumentos/Retorno:
  • arg - Data ou diferenca de tempo
  • return - ano da data ou quantidade de anos
Exemplo:
object data = date( "15/05/2014 12:30:45" )
object hoje = date( "01/04/2014 15:30:15" )
object tempo = diff( hoje, data )
print( getyear( data ) )
print( getyear( tempo ) )
