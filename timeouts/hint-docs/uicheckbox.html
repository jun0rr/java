Funcoes de Ambiente Gráfico
uicheckbox( string text, string onsel )
Cria um elemento gráfico do tipo caixa de seleção (check box).
Argumentos/Retorno:
  • text - Texto a ser exibido ao lado da caixa de seleção
  • onsel - Função chamada quando o elemento for selecionado (opcional)
Exemplo:
object check = uicheckbox( "Select me" )
uisetselected( check, "1" )
