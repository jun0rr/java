Funcoes de Data
gethour( object arg )
Retorna a hora do dia da data informada, ou a quantidade de horas da diferenca de tempo.
Argumentos/Retorno:
  • arg - Data ou diferenca de tempo
  • return - hora do dia ou quantidade de horas
Exemplo:
object data = date( "15/04/2014 12:00:00" )
object hoje = date( "01/04/2014 15:00:00" )
object tempo = diff( hoje, data )
print( gethour( data ) )
print( gethour( tempo ) )
