Funcoes de Texto
contains( string arg1, string arg2 )
Verfica se o primeiro texto (arg1) contem o segundo argumento (arg2), retornando verdadeiro ou falso.
Argumentos/Retorno:
  • arg1 - Texto original
  • arg2 - Texto a ser pesquisado
  • return - Verdadeiro/Falso
Exemplo:
string texto = "Algum Texto"
if contains( texto, "Algum" )
   print( "A variavel TEXTO contem 'Algum'!" )
endif
# Imprime no console
> A variavel TEXTO contem 'Algum'!
