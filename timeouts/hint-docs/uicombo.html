Funcoes de Ambiente Gráfico
uicombo( string onselect, string ... elements )
Cria um elemento gráfico do tipo caixa de combo (combo box).
Argumentos/Retorno:
  • onselect - Função a ser chamada quando houver seleção
  • elements - Lista de elementos a ser exibidos na caixa de combo, separados por vírgula
Exemplo:
object combo = uicombo( "", "um", "dois", "tres" )
uisetselected( combo, "dois" )
