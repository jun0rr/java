Welcome to PowerStat - Monitor and Diagnostic System

     With the CPU Application, is possible to monitor de CPU consumption with I/O wait, user and system load, besides allows to viewing processor information and generates custom historical charts.


     The storage devices can be monitored by the Disk Application, which allow to generate consumption charts and display information about the mounting point and file system.


     The running processes are valueable information for the system administrator, and can be real-time monitored by the Processes Application, allowing to apply filters and sort order, with the power to kill processes.


     The main memory is one of most critical components in servers and the Memory Application allows to monitor current and historical consumption, and the possibility to clear memory cache, allowing to retrieve around 25% of the consumed memory.


     The Network Application allow real-time monitoring to the network interfaces and can generate custom historical charts, besides can display all traffic for the connected ip addresses, with protocol names and port number.


     The most powerfull tool for a system administrator is the terminal, which can be accessed by the web interface, allowing full control of the system with easy access.
