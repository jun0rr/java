Funcoes de Texto
eq( string arg1, string arg2 )
Compara o conteudo de dois textos (strings) e retorna verdadeiro se forem iguais, ou falso caso contratio.
Argumentos/Retorno:
  • arg1 - Texto1 a ser comparado
  • arg2 - Texto2 a ser comparado
  • return - Verdadeiro/Falso
Exemplo:
string txt = "cafe"
if eq( txt, "cafe" )
   print( "Variavel txt = 'cafe'!" )
endif
# Imprime no console
> Variavel txt = 'cafe'!
