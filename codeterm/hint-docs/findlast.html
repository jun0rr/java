Funcoes de Texto
findlast( string arg1, string arg2 )
Encontra o texto do argumento arg2 no texto do argumento arg1, retornando o numero da posicao (comecando em zero) da ultima ocorrencia, de arg2, ou -1 se nao for encontrado.
Argumentos/Retorno:
  • arg1 - Texto no qual sera efetuada a pesquisa
  • arg2 - Texto a ser encontrado
  • return - Numero da posicao, ou -1 se nao for encontrado
Exemplo:
# ''''''''''''0''''5'''10'''15'''''''24
string txt = "algum texto sem sentido algum"
int pos = findlast( txt, "algum" )
print( "Ultima posicao de 'algum' = " + pos )
# Imprime no console
> Ultima posicao de 'algum' = 24
