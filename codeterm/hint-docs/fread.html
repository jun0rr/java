Funcoes de Leitura/Escrita (I/O)
fread( string arq, string delim )
Le o conteudo de texto do arquivo arq ate encontrar o delimitador delim. Se nenhum delimitador for fornecido, e assumido a quebra de linha LN como delimitador.
Argumentos/Retorno:
  • arq - Nome do arquivo
  • delim - Delimitador ate onde sera efetuada a leitura do arquivo
  • return - Texto lido do arquivo
Exemplo:
string arquivo = "c:/dados.txt"
string coluna = fread( arquivo, CSV )
