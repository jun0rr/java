Funcoes de Ambiente Gráfico
uibutton( string title, string onclick )
Cria um botão gráfico de ação.
Argumentos/Retorno:
  • title - Texto a ser exibido no botão
  • onclick - Função a ser chamada quando o botão for clicado
Exemplo:
int nr = 0 func onclick() nr = nr + 1 print("O botão foi clicado "+ nr+ " vezes.") endfunc object button = uibutton( "Click me", "onclick" )
