Funcoes de Data
after( object date1, object date2 )
Compara duas datas, retornando verdadeiro se a 1a data for posterior a 2a, ou retorna falso caso contrario.
Argumentos/Retorno:
  • date1 - Data a ser comparada
  • date2 - Data a ser comparada
  • return - Verdadeiro/falso
Exemplo:
object hoje = date( "01/04/2014 12:00:00" )
object amanha = addday( hoje, 1 )
if after( amanha, hoje )
    print( "Amanha vem depois de hoje!" )
endif
# Imprime no console
> Amanha vem depois de hoje!
