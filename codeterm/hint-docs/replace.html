Funcoes de Texto
replace( string text, string arg, string rep )
Substitui todas as ocorrencias de arg na string text pelo conteudo do argumento rep.
Argumentos/Retorno:
  • text - Texto no qual ocorrerao as substituicoes
  • arg - Texto a ser substituido
  • rep - Texto pelo qual arg sera substituido
  • return - Texto com as substituicoes efetuadas
Exemplo:
string cpf = "123.456.789-00"
cpf = replace( cpf, ".", "" )
print( "cpf = " + cpf )
# Imprime no console
> cpf = 123456789-00
