Funcoes de Banco de Dados (MySQL)
mysql_connect( string addr, [string schema], string user, string pwd )
Inicia uma conexão com servidor MySQL no endereço especificado, autenticando com o usuário e senha fornecidos.
O argumento schema (opcional), seleciona o schema do banco de dados que será utilizado na conexão.
Retorna o objeto Connection que representa a conexão estabelecida.
Argumentos/Retorno:
  • addr - Endereço IP ou nome do servidor
  • [schema] - Schema a ser usado na conexão (opcional)
  • user - Usuário do banco de dados
  • pwd - Senha de autenticação
  • return - Objecto Connection que representa a conexão
Exemplo:
# Inicia uma conexão na máquina local
object conn = mysql_connect( "127.0.0.1", "usuario", "senha" )
