Funcoes de Conversao de Tipos
toint( object arg )
Tenta converter o argumento arg em numero, retornando o numero convertido ou -1 em caso de erro.
Argumentos/Retorno:
  • arg - Argumento a ser convertido
  • return - Numero convertido ou -1 em caso de erro
Exemplo:
string txt = "500"
int nr = toint( txt )
