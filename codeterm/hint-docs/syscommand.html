Funcoes de Leitura/Escrita (I/O)
syscommand( string cmd )
Executa o comando cmd no sistema operacional, retornando a saida do comando como string.
Argumentos/Retorno:
  • cmd - Comando a ser executado
  • return - Saida da execucao do comando no sistema
Exemplo:
string comando = "cmd /c dir G:"
string saida = syscommand( comando )
print( saida )
