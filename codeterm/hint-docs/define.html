Funcoes de Recursos da Linguagem
define( string name, [object value] )
Define uma variável global que pode receber qualquer tipo de valor e pode ser referenciada em qualquer lugar do código.
Argumentos/Retorno:
  • name - Nome da variável global
  • value - (Opcional) Valor da variável
Exemplo:
# Cria uma variável global sem valor.
# Variáveis sem valor não podem ser referenciadas antes de receberem um valor
define( "variavel" )
# Atribui um objeto de data à variável.
object data = date( )
define( "variavel", data )
print( variavel )
