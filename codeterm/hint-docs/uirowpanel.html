Funcoes de Ambiente Gráfico
uirowpanel( int vgap )
Cria um painel para agregação e organização de outros elementos gráficos. Os elementos adicionados são organizados em linhas dispostas de forma vertical, uma abaixo da outra de acordo com a ordem que forem adicionadas.
Argumentos/Retorno:
  • vgap - Distância horizontal entre os elementos, em pixels
Exemplo:
object label = uilabel( "Nome:" )
object text = uitext( "", 150 )
object rpanel = uirowpanel( 10 )
uiadd( rpanel, label )
uiadd( rpanel, text )
