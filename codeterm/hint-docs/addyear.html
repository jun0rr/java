Funcoes de Data
addyear( object date, int nr )
Acrescenta nr anos na data date.
Argumentos/Retorno:
  • date - Data que sera adiantada
  • nr - Numero de anos acrescidos
  • return - Nova data acrescida em nr anos
Exemplo:
object hoje = date( "01/04/2014 12:00:00" )
object ano2015 = addyear( hoje, 1 )
print( "2015 sera " + ano2015 )
# Imprime no console
> 2015 sera 01/04/2015 12:00:00
