Funcoes de Leitura/Escrita (I/O)
fwimage( string arq, object img )
Escreve a imagem img no arquivo arq em formato JPEG.
Argumentos:
  • arq - Nome do arquivo
  • img - Objeto Imagem a ser escrito
Exemplo:
string arquivo = "c:/tela.jpg"
object image = screencapture( 0, 0, 800, 600 )
fwimage( arquivo, image )
