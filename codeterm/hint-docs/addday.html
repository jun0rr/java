Funcoes de Data
addday( object date, int nr )
Acrescenta nr dias na data date.
Argumentos/Retorno:
  • date - Data que sera adiantada
  • nr - Numero de dias acrescidos
  • return - Nova data acrescida em nr dias
Exemplo:
object hoje = date( "31/03/2014" )
object amanha = addday( hoje, 1 )
print( "Amanha sera " + amanha )
# Imprime no console
> Amanha sera 01/04/2014 00:00:00
