Funcoes SISBB
find( string txt )
Tenta encontrar o texto txt na tela do terminal, retornando um objeto cursor representando a posicao indicada.
Argumentos/Retorno:
  • txt - Texto a ser encontrado na tela
  • return - Objeto cursor
Exemplo:
object cursor = find( "Pessoal" )
int linha = cursor.row()
int coluna = cursor.column()
if cursor.position() == 0
    # Texto nao encontrado
endif
