Funcoes de Banco de Dados (MySQL)
mysql_next( object result )
Move o cursor para a próxima linha no resultado.
Argumentos/Retorno:
  • result - Resultado de consulta no banco de dados
  • return - Verdadeiro(1)/Falso(0) indicando se existe próxima linha no resultado.
Exemplo:
object result = ...
while mysql_next( result )
   ...
endwhile
