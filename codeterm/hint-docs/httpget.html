Funcoes de Rede
httpget( string addr, int port, [string proxyAddr], [int proxyPort], [string proxyAuth] )
Executa uma requisição HTTP para um servidor web, retornando o objeto Socket que representa a conexão.
Argumentos/Retorno:
  • addr - Endereço do servidor web
  • port - Porta do servidor web
  • proxyAddr - (Opcional) Endereço do servidor proxy da rede
  • proxyPort - (Opcional) Porta do servidor proxy da rede
  • proxyAuth - (Opcional) Autenticação para o servidor proxy (Formato '<usuário>:<senha>')
Exemplo:
# Inicia uma requisição para a página do Google
object socket = httpget( "http://www.google.com.br", 80 )
# Lê a resposta do servidor
string linha = tcpreadln( socket )
# Enquanto a operação de leitura não retornar o sinal de fim de arquivo, imprime a linha na tela
while ! eq( EOF, linha )
   print( linha )
endwhile
# Encerra a conexão
tcpclose( socket )
