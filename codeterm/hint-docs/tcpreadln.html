Funcoes de Rede
tcpreadln( object socket )
Lê uma linha de texto do socket da conexão.
Argumentos/Retorno:
  • socket - Socket que representa a conexão
  • return - A linha de texto lida ou o sinal de fim de arquivo EOF
Exemplo:
# Inicia uma requisição para a página do Google
object socket = httpget( "http://www.google.com.br", 80 )
# Lê a resposta do servidor
string linha = tcpreadln( socket )
# Enquanto a operação de leitura não retornar o sinal de fim de arquivo, imprime a linha na tela
while ! eq( EOF, linha )
   print( linha )
endwhile
# Encerra a conexão
tcpclose( socket )
