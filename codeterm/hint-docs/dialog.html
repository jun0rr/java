Funcoes de Ambiente Grafico
dialog( int tipo, string titulo, string mensagem )
Exibe uma janela/dialogo de informacoes. O tipo de janela e definido pelas variaveis globais iniciadas por DL.
Argumentos:
  • tipo - Tipo de janela (Informacao, Alerta, Erro, Pergunta)
  • titulo - Titulo da janela
  • mensagem - Mensagem a ser exibida
Exemplo:
string titulo = "Titulo da janela"
string txt = "Processamento Concluido" dialog( DLINFO, titulo, txt )

titulo = "Titulo"
txt = "Informe seu nome"
string nome = dialog( DLINPUT, titulo, txt )
print( nome )

titulo = "Pergunta"
txt = "Voce e maior de idade?"
string resp = dialog( DLQUESTION, titulo, txt )
if eq( resp, YES )
    print( ">= 18" )
else
    print( "< 18" )
endif
