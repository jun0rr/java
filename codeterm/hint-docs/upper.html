Funcoes de Texto
upper( string arg )
Converte um texto (string) em letras maiusculas.
Argumentos/Retorno:
  • arg - Texto a ser convertido em letras maiusculas
  • return - Texto em letras maiusculas
Exemplo:
string txt = "letras minusculas"
txt = lower( txt )
print( txt )
# Imprime no console
> LETRAS MINUSCULAS
