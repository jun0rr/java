Funcoes de Rede
tcpread( object socket, [int size] )
Lê conteúdo binário do socket da conexão.
Se o argumento size for informado, lê e retorna uma coleção de bytes da conexão com o tamanho informado, caso contrário retorna apenas um byte.
Argumentos/Retorno:
  • socket - Socket que representa a conexão
  • size - (Opcional) Tamanho da coleção de bytes a ser lida
  • return - byte/coleção de bytes lido(s) ou sinal de fim de arquivo EOF
Exemplo:
object socket = ...
# Lê o conteúdo binário do socket da conexão
object bytes = tcpread( socket, 1024 )
# Faz algo com o conteúdo lido
... # Encerra a conexão
tcpclose( socket )
