Funcoes de Texto
str( object arg )
Converte o argumento em texto (string).
Argumentos/Retorno:
  • arg - objeto a ser convertido em texto
  • return - Texto (string) representando o argumento
Exemplo:
int x = 500
string s = "500"
if eq( s, str( x ) )
    print( "'500' = " + x )
endif
# Imprime no console
> '500' = 500
