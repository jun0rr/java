Funcoes SISBB
select( int lin, int col, int larg, int alt )
Seleciona uma parte da tela do terminal, iniciando na posicao indicada por lin e col, com o comprimento larg e altura alt.
Argumentos:
  • lin - Linha da posicao do terminal
  • col - Coluna da posicao do terminal
  • larg - Largura da selecao em numero de caracteres
  • alt - Altura da selecao em numero de linhas
Exemplo:
select( 1, 3, 10, 1 )
