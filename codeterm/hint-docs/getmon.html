Funcoes de Data
getmon( object arg )
Retorna o mes da data informada, ou a quantidade de meses da diferenca de tempo.
Argumentos/Retorno:
  • arg - Data ou diferenca de tempo
  • return - mes da data ou quantidade de meses
Exemplo:
object data = date( "15/05/2014 12:30:45" )
object hoje = date( "01/04/2014 15:30:15" )
object tempo = diff( hoje, data )
print( getmon( data ) )
print( getmon( tempo ) )
