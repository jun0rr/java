Funcoes de Ambiente Gráfico
uiradio( string onselect, string ... names )
Cria um grupo de elementos gráficos do tipo grupo de botões de rádio (radio button), onde apenas um botão do grupo pode ser selecionado por vez.
Argumentos/Retorno:
  • onselect - Função a ser chamada quando houver seleção
  • names - Nomes dos botões de rádio do grupo
Exemplo:
func select( object sel )
   print( "Objeto selecionado: "+ sel )
endfunc
object radio = uiradio( "select", "um", "dois", "tres" )
uisetselected( radio, "dois" )
# Imprime no console
> Objeto selecionado: dois
