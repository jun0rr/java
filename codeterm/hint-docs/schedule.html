Funcoes de Agendamento de Tarefas
schedule( object date, string func )
Agenda a execucao da funcao func previamente definida.
Argumentos:
  • date - Data de execucao da funcao
  • func - Nome da funcao a ser executada
Exemplo:
object hoje = date( "15/05/2014 12:00:00" )
func imprime()
    print( "Executando a funcao 'imprime'" )
endfunc
schedule( hoje, "imprime" )
