Funcoes de Texto
startswith( string str1, string str2 )
Verifica se o texto do primeiro argumento começa com o texto do segundo argumento.
Argumentos/Retorno:
  • str1 - Texto1 a ser verificado
  • str2 - Texto2 a ser comparado
  • return - Verdadeiro/Falso
Exemplo:
string txt = "cafe quente"
if startswith( txt, "cafe" )
   print( "Variavel txt começa com 'cafe'!" )
endif
# Imprime no console
> Variavel txt começa com 'cafe'!
