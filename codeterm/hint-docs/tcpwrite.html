Funcoes de Rede
tcpwrite( object socket, int byte/object bytes )
Escreve o conteúdo binário no socket da conexão.
Argumentos/Retorno:
  • socket - Socket que representa a conexão
  • byte/bytes - byte/coleção de bytes a serem escritos no socket
Exemplo:
object socket = ...
object bytes = ...
# Escreve o conteúdo binário no socket da conexão
tcpwrite( socket, bytes )
# Encerra a conexão
tcpclose( socket )
