Funcoes de Leitura/Escrita (I/O)
frbytes( string arq, int nbytes )
Le o conteudo binario do arquivo arq. O segundo argumento nbytes define a quantidade de bytes que devem ser lidos. A funcao retorna um objeto que contem uma colecao (array) dos bytes lidos.
Argumentos/Retorno:
  • arq - Nome do arquivo
  • nbytes - Quantidade de bytes a serem lidos
  • return - Array contendo os bytes lidos
Exemplo:
string arquivo = "c:/programa.exe"
object conteudo = frbytes( arquivo, 4096 )
# Efetua o tratamento do conteudo lido...
