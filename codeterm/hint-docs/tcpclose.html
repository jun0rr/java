Funcoes de Rede
tcpclose( object obj )
Encerra uma conexão TCP/IP do cliente/servidor informado.
Todas as conexões, sejam de cliente ou servidor, devem ser encerradas após o uso para liberar recursos de máquina.
Argumentos/Retorno:
  • obj - Objeto que representa uma conexão cliente (Socket) ou um servidor
Exemplo:
# Inicia uma conexão na máquina local (localhost) na porta 8080
object socket = tcpconnect( "127.0.0.1", 8080 )
# Encerra a conexão previamente iniciada
tcpclose( socket )
