Funcoes de Banco de Dados (MySQL)
mysql_column( object result, object column )
Obtém o valor contida na coluna informada.
O argument column pode ser o número ou nome da coluna.
Argumentos/Retorno:
  • result - Resultado de consulta no banco de dados
  • column - Número ou nome da coluna cujo valor será obtido
  • return - Valor da coluna
Exemplo:
object result = ...
object valor = mysql_column( result, 1 )
# object valor = mysql_column( result, "nomeColuna" )
