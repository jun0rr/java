Funcoes de Data
dayofw( object date )
Retorna o dia da semana da data informada, sendo que 1 (um) representa domingo e 7 (sete) representa sabado.
Argumentos/Retorno:
  • date - data a ser verificado o dia da semana
  • return - 1 a 7 representando o dia da semana.
Exemplo:
object hoje = date( "01/04/2014 12:00:00" )
int day = dayofw( hoje )
print( "terca-feira = " + day )
# Imprime no console
> terca-feira = 3
