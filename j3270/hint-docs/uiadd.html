Funcoes de Ambiente Gráfico
uiadd( object cont, object c )
Adiciona um elemento gráfico à outro elemento de agregação, como um painel ou janela.
Argumentos/Retorno:
  • cont - Elemento container que irá agregar outro elemento gráfico
  • c - Elemento gráfico a ser adicionado
Exemplo:
object label = uilabel( "Nome:" )
object text = uitext( "", 150 )
object rpanel = uirowpanel( 10 )
uiadd( rpanel, label )
uiadd( rpanel, text )
