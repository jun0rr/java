Funcoes de Data
diff( object date1, object date2 )
Retorna a diferenca de tempo entre duas datas.
Argumentos/Retorno:
  • date1 - Primeira data
  • date2 - Segunda data
  • return - Diferenca de tempo entre as datas.
Exemplo:
object hoje = date( "01/04/2014 12:00:00" )
object amanha = date( "02/04/2014 14:00:00" )
object tempo = diff( hoje, amanha )
print( "diferenca = " + tempo )
# Imprime no console
> diferenca = 1 day(s), 2:0:0.0
