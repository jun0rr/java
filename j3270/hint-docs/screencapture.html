Funcoes de Ambiente Grafico
screencapture( int x, int y, int larg, int alt )
Captura a tela do monitor, iniciando no ponto definido por x e y, com a largura e altura indicados por larg e alt.
Argumentos:
  • x - Posicao horizontal do click na tela
  • y - Posicao vertical do click na tela
  • larg - Largura da imagem a capturar, em pixels
  • alt - Altura da imagem a capturar, em pixels
Exemplo:
object img1 = screencapture( 0, 0, 800, 600 )
object img2 = screencapture( 0, 0, SCREENW, SCREENH )
