Funcoes SISBB
waitelse( int lin, int col, string txt, int lin2, int col2, string txt2 )
Aguarda a atualizacao do terminal e o texto txt na posicao indicada por lin e col. Caso o texto txt nao esteja definido na posicao indicada apos a atualizacao do terminal, aguarda pelo segundo texto txt2 na posicao lin2 e col2. Se nenhuma condicao for atendida, um erro e gerado.
Argumentos/Retorno:
  • lin - Linha da posicao do terminal
  • col - Coluna da posicao do terminal
  • txt - Texto que devera aparecer apos a atualizacao do terminal
  • lin2 - Linha da segunda posicao no terminal
  • col2 - Coluna da segunda posicao no terminal
  • txt2 - Texto que devera aparecer no caso de falha do primeiro
  • return - Um dos dois textos a ser exibido no terminal apos a atualizacao
Exemplo:
string t1 = "BB30"
string t2 = "IBBM"
string id = waitelse( 1, 2, t1, 1, 3, t2 )
if eq( id, t2 )
    # Tela de noticias
    f3()
endif
