Funcoes de Banco de Dados (MySQL)
mysql_col_string( object result, object column )
Obtém o valor do tipo string (texto) contido na coluna informada.
O argument column pode ser o número ou nome da coluna.
Argumentos/Retorno:
  • result - Resultado de consulta no banco de dados
  • column - Número ou nome da coluna cujo valor será obtido
  • return - Valor (string) da coluna
Exemplo:
object result = ...
string valor = mysql_col_string( result, 1 )
# string valor = mysql_col_string( result, "nomeColuna" )
