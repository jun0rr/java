Funcoes SISBB
setpassword( int lin, int col, string senha )
Define uma senha codificada em Base64 na posicao indicada no terminal.
Argumentos:
  • lin - Linha da posicao do terminal
  • col - Coluna da posicao do terminal
  • senha - Senha codificada em Base64
Exemplo:
string senha = "MTIzNDU2Nzg="
setpassword( 16, 14, senha )
