Funcoes de Data
getmin( object arg )
Retorna os minutos da hora da data informada, ou a quantidade de minutos da diferenca de tempo.
Argumentos/Retorno:
  • arg - Data ou diferenca de tempo
  • return - minutos da hora ou quantidade de minutos
Exemplo:
object data = date( "15/04/2014 12:30:00" )
object hoje = date( "01/04/2014 15:30:00" )
object tempo = diff( hoje, data )
print( getmin( data ) )
print( getmin( tempo ) )
