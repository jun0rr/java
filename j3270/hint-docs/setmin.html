Funcoes de Data
setmin( object date, int min )
Define os minutos da data informada, retornando a mesma.
Argumentos/Retorno:
  • data - Data
  • min - Minuto da hora a ser definido
  • return - Data com os minutos definidos
Exemplo:
object data = date( "15/05/2014" )
data = setmin( data, 25 )
print( data )
