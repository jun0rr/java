Funcoes de Ambiente Gráfico
uisetselected( object comp, string elto )
Define o elemento selecionado em um elemento de seleção.
Argumentos/Retorno:
  • comp - Elemento gráfico de seleção
  • elto - Elemento selecionado
Exemplo:
object combo = uicombo( "", "one", "two", "ten" )
uisetselected( combo, "two" )
