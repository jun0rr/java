Funcoes de Leitura/Escrita (I/O)
fwbytes( string arq, object bytes )
Escreve o conteudo binario bytes no arquivo arq.
Argumentos:
  • arq - Nome do arquivo
  • bytes - Conteudo a ser escrito
Exemplo:
string leitura = "c:/arquivo.exe"
string escrita = "c:/copia.exe"
object conteudo = frbytes( leitura, 4096 )
fwbytes( escrita, conteudo )
