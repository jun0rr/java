PowerStat - Quick Start

Use the keyboard keys (left) and (right) to navigate between pages.


To install and run PowerStat, you must check some dependencies, have sure you met them all before start:

PowerStat - 1 / 19

• A Linux server running;
• Java environment installed and configured;
• Unlimited Cryptography Extension (JCE)
Unlimited Strength Jurisdiction Policy Files for
Java, properly configured;
• Apache Tomcat installed and configured;
• jnettop installed (apt-get install jnettop);
• ifstat installed (apt-get install ifstat);

PowerStat - 2 / 19

Download PowerStat binary and point your browser to Apache Admin Page .

PowerStat - 3 / 19

On Tomcat Admin page, Click on "Manager App" button.

PowerStat - 4 / 19

In "Deploy" session, click on "Browse" button.

PowerStat - 5 / 19

Select PowerStat binary file (.war) and then click on "Deploy" button.

PowerStat - 6 / 19

Point your browser to the the address:

http://localhost:8080/pstat/index.xhtml.

PowerStat - 7 / 19

The PowerStat initial page must be displayed. Click on "LOGIN" link.

PowerStat - 8 / 19

The default login is admin@localhost and password is "admin".

PowerStat - 9 / 19

When "Configure PowerStat" dialog apear, click on "Yes" button.

PowerStat - 10 / 19

The Configuration page is displayed. Configure the options of the Syscheck module and save changes.

PowerStat - 11 / 19

Check the "Run As Sudo" box, input the password and click on "Install Module" button.

PowerStat - 12 / 19

Now the Syscheck module is installed. Click on "Start Syscheck" button and wait for "Running" status apear.

PowerStat - 13 / 19

Just click "OK" to register a new admin user.

PowerStat - 14 / 19

On the User Admin page, click on "Create New User".

PowerStat - 15 / 19

Inform user name, e-mail, password and click "Save".

PowerStat - 16 / 19

Configure the new user.

Check the "Admin" box and add some permissions ("ALL" for an admin user is recommended).

PowerStat - 17 / 19

Dont forget to "Save Modifications".

PowerStat - 18 / 19

It's Done!
